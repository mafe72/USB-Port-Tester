Basic Usage
-----------

* Green LED ON
			
		USB Port Polarity OK
		
* Red LED ON
		
		USB Port Polarity Reversed
		
* Blue LED ON/FLICKERS
		
		USB Port DATA Port OK

* NO LED ON
		
		USB Port DEAD OR DISCONNECTED
